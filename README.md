# One Another by Koinonia

React + Vite prototype.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Go to [https://vercel.com/new](https://vercel.com/new).
3. Import the repo.
4. Deploy.

This repo includes `vercel.json` configured for Vite.
